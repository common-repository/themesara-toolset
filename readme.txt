== Description ==
Import <a href="https://www.themesara.com/themes/" target="_blank" rel="nofollow">ThemeSara</a> official themes demo content, widgets and theme settings with just one click.

Get [free support](https://www.themesara.com/supports/)

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for themesara Toolkit
3. Activate Themesara Toolkit plugin from your Plugins page.
4. Go to the Appearance -> Demo Data

= From WordPress.org =

1. Download themesara toolkit.
2. Upload the 'themesara-toolkit' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate themesara Toolkit from your Plugins page.
4. Go to the Customizer (Appearance -> Customize) to add custom color to your site.
